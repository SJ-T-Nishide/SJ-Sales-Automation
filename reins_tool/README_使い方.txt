■ フォルダ構成
────────────────────────────────────────────────────────────────────
C:\reins_tool\
  ├─ config.ini            ← 【設定ファイル】ここを編集してください
  ├─ reins_rpa_v4.ahk      ← AHK 自動操作スクリプト
  ├─ reins_to_tsv.py       ← Python TSV 変換スクリプト
  ├─ run_reins.bat         ← 【メイン起動】ダブルクリックで全自動実行
  ├─ run_convert.bat       ← TSV 変換のみ単体実行（手動変換時）
  ├─ input\                ← reins_*.txt を手動で置く場合はここへ
  ├─ output\               ← TSV・エラーログ出力先
  └─ log\                  ← AHK 操作ログ出力先

※ input・output・log フォルダは初回実行時に自動作成されます。
────────────────────────────────────────────────────────────────────


■ 前提条件（インストールが必要なソフト）
────────────────────────────────────────────────────────────────────
1. Python 3.9 以上
   ダウンロード: https://www.python.org/downloads/
   ※ インストール時に「Add Python to PATH」にチェックを入れること

2. pyperclip（Pythonライブラリ）
   インストール: コマンドプロンプトで以下を実行
   > pip install pyperclip

3. AutoHotkey v2
   ダウンロード: https://www.autohotkey.com
   ※ v2 系をインストールしてください（v1 では動作しません）
────────────────────────────────────────────────────────────────────


■ 初期設定
────────────────────────────────────────────────────────────────────
config.ini をメモ帳などで開き、必要に応じて編集してください。

  [paths]
  input_folder  = C:\reins_tool\input    ← 引数なし起動時の入力フォルダ
  output_folder = C:\reins_tool\output   ← TSV 出力先
  log_folder    = C:\reins_tool\log      ← AHK ログ出力先

  [options]
  copy_to_clipboard = true   ← false にするとクリップボードコピーをしない

  [tabs]
  tab_01 = 01 大阪_一戸建    ← 条件番号とスプレッドシートのシート名の対応
  tab_02 = 02 大阪市_マンション
  ... （変更する場合はシート名と完全一致させること）

注意: config.ini は UTF-8（BOMなし）で保存してください。
      メモ帳の場合、「名前を付けて保存」→ 文字コード「UTF-8」を選択。
────────────────────────────────────────────────────────────────────


■ 使い方 ― 全自動モード（通常の使い方）
────────────────────────────────────────────────────────────────────
1. REINS にログインし、「GBK001310（保存条件選択画面）」を開いておく
   URL: https://system.reins.jp/main/BK/GBK001310

2. C:\reins_tool\run_reins.bat をダブルクリック

3. 3秒後に AHK が自動起動し、条件01〜20 を順番に取得開始
   ※ 所要時間: 約20〜30分
   ※ 取得中はマウス・キーボードを触らないこと

4. 取得完了後、自動で TSV 変換が実行される
   出力: C:\reins_tool\output\converted_YYYYMMDD_HHMM.tsv

5. TSV をコピーして Google スプレッドシートの「受信」タブに貼り付け

6. スプレッドシートのメニュー「🏠 民泊DB」→「受信データを振り分ける」
────────────────────────────────────────────────────────────────────


■ 使い方 ― ホットキー（手動モード）
────────────────────────────────────────────────────────────────────
AHK 起動後、以下のホットキーで部分実行できます:

  Ctrl+1  条件01のみテスト実行
  Ctrl+2  条件01〜05 を実行
  Ctrl+3  全20条件を実行（確認ダイアログあり）
  Ctrl+0  停止・終了
────────────────────────────────────────────────────────────────────


■ 使い方 ― TSV 変換のみ単体実行
────────────────────────────────────────────────────────────────────
すでに reins_*.txt ファイルがある場合:

  方法A: run_convert.bat に reins_*.txt をドラッグ＆ドロップ

  方法B: コマンドプロンプトで実行
  > cd C:\reins_tool
  > python reins_to_tsv.py C:\reins_tool\input\reins_20260501_1030.txt

  方法C: 引数なし（input フォルダの最新ファイルを自動選択）
  > python reins_to_tsv.py
────────────────────────────────────────────────────────────────────


■ 別の PC へのセットアップ手順
────────────────────────────────────────────────────────────────────
1. C:\reins_tool\ フォルダごとコピー（USB やネットワーク経由など）

2. 新しい PC で前提条件（Python + pyperclip + AHK v2）をインストール

3. 必要なら config.ini のフォルダパスを修正

4. run_reins.bat をダブルクリックして動作確認
────────────────────────────────────────────────────────────────────


■ トラブルシューティング
────────────────────────────────────────────────────────────────────
Q. 「AutoHotkey が見つかりません」と表示される
   → AHK v2 をインストールし、PATH に登録されているか確認。
     または C:\Program Files\AutoHotkey\v2\ にあるか確認。

Q. 「Python not found」と表示される
   → Python インストール時に「Add Python to PATH」が有効か確認。
     有効でない場合はインストールし直すか PATH を手動で追加。

Q. クリップボードにコピーされない
   → config.ini の copy_to_clipboard = true を確認。
     または pip install pyperclip を実行。

Q. TSV の文字が化ける
   → TSV ファイルは UTF-8 BOM ありで保存されています。
     スプレッドシートへの貼り付けは、ブラウザで開いた Google スプレッドシートに
     直接貼り付けるか、テキストエディタで開いて内容をコピーしてください。

Q. 条件取得中に止まった・エラーになった
   → log フォルダのログファイルで最後に処理した条件を確認。
     REINS の画面がログアウトしていないか確認して再実行してください。
