# Plaud AI専用UI 設計書

**作成日:** 2026-05-23  
**アーキテクチャ:** Google Apps Script + Claude API  
**Phase:** 4  
**セキュリティレベル:** 最高（社長のみアクセス）

---

## 📌 システム概要

### 目的
社長が個人的に録音した音声（Plaud AI）の文字起こしを分析し、経営判断に必要なタスク・アイデア・シグナルを抽出する。

### プライバシー設計思想
1. **元の文字起こしは社長のみが閲覧可能** - チームには共有しない
2. **抽出されたタスク・アイデアのみを選択的に共有** - 社長が承認したもののみ
3. **すべてのデータは社長のGoogle Drive内で管理** - 外部流出を防ぐ

---

### 2. データ保存構造

#### Google Sheets: Plaud_文字起こし（社長専用）

| 列 | 内容 | 権限 |
|----|------|------|
| A: タイムスタンプ | 録音日時 | private |
| B: カテゴリ | 経営判断/顧客面談/アイデアメモ/etc | private |
| C: 文字起こし全文 | Plaud AIの元テキスト | **private（社長のみ閲覧可能）** |
| D: 分析結果JSON | Claude APIの出力 | private |
| E: 共有ステータス | 「未共有」「一部共有」「全て共有」 | private |

**アクセス権限:**
- 所有者: 社長のGoogleアカウント
- 共有設定: なし（社長のみ）

#### Google Sheets: AI経営シグナル_データベース（チーム共有）

**シート: Plaud抽出タスク**

| 列 | 内容 | 権限 |
|----|------|------|
| A: タイムスタンプ | タスク抽出日時 | internal |
| B: タスク内容 | 具体的なタスク | internal |
| C: 担当者 | アサイン先 | internal |
| D: 期限 | 期限日 | internal |
| E: 重要度 | S/A/B/C | internal |
| F: ステータス | 未着手/進行中/完了 | internal |
| G: 元ソース | 「Plaud分析」（元テキストは含まない） | internal |

**アクセス権限:**
- 所有者: 社長
- 共有: 全社員（閲覧・編集可）

### 3. Claude API分析プロンプト

```markdown
あなたは経営者の音声メモ分析AIです。
以下の文字起こしテキストから、経営判断に必要な情報を抽出してください。

【文字起こしテキスト】
{文字起こし全文}

【抽出項目】

1. タスク
具体的なアクションが必要な項目を抽出。
- タスク内容（簡潔に）
- 期限（明示されている場合）
- 担当者（推測可能な場合）
- 重要度（S/A/B/C）
- 推奨共有先（営業部/管理部/コーポレート/共有不要）

2. アイデア
新規事業・商材・改善案などを抽出。
- アイデア内容
- カテゴリ（新商材/業務改善/市場機会/その他）
- 推奨共有先

3. 重要シグナル
市況変化・顧客動向・社内課題などを抽出。
- シグナル内容
- 種別（市況/顧客/社内/その他）
- 重要度（S/A/B/C）
- 推奨共有先

【営業トーク判定】
顧客面談での撤退・損失の話は、営業トークの可能性を考慮して重要度を判定してください。
実数データがない場合は重要度を1段階下げてください。

【出力フォーマット】
JSON形式で出力してください。
{
  "tasks": [
    {
      "content": "タスク内容",
      "deadline": "YYYY-MM-DD" or null,
      "assignee": "担当者" or null,
      "priority": "S/A/B/C",
      "share_with": "営業部/管理部/コーポレート/共有不要"
    }
  ],
  "ideas": [
    {
      "content": "アイデア内容",
      "category": "新商材/業務改善/市場機会/その他",
      "share_with": "営業部/管理部/コーポレート/共有不要"
    }
  ],
  "signals": [
    {
      "content": "シグナル内容",
      "type": "市況/顧客/社内/その他",
      "priority": "S/A/B/C",
      "share_with": "営業部/管理部/コーポレート/共有不要"
    }
  ]
}
```

## 📝 実装手順

### Phase 4B: Plaud専用UI実装（1週間）

**1. Google Sheets準備（30分）**

a. 社長のGoogle Driveに新規スプレッドシート作成
   - 名前: `Plaud_文字起こし`
   - 共有設定: 社長のみ
   
b. シート構成
   - シート1: `文字起こし履歴`（A-E列）
   - シート2: `分析結果`（JSON保存）

**2. Google Apps Scriptプロジェクト作成（3-4時間）**

a. スクリプトエディタを開く
   - スプレッドシート → 「拡張機能」→ 「Apps Script」
   
b. ファイル構成
   ```
   PlaudAnalysisApp/
   ├── Code.gs           # メインロジック
   ├── ClaudeAPI.gs      # Claude API連携
   ├── UI.html           # WebアプリUI（画面1）
   ├── Result.html       # 分析結果画面（画面2）
   └── Stylesheet.html   # CSS
   ```

c. スクリプトプロパティに設定
   - `CLAUDE_API_KEY`: Claude APIキー
   - `OWNER_EMAIL`: 社長のGmailアドレス
   - `SHARED_SPREADSHEET_ID`: AI経営シグナル_データベースのID

**3. Webアプリデプロイ（1時間）**

a. デプロイ設定
   - 「デプロイ」→ 「新しいデプロイ」
   - 種類: Webアプリ
   - アクセス権限: **自分のみ**（社長のGoogleアカウントのみ）
   
b. URLを取得
   - 例: `https://script.google.com/macros/s/AKfycbz.../exec`

**4. Claude API統合（2-3時間）**

GASでClaude APIを呼び出し。

```javascript
function analyzePlaudText(text, category) {
  const apiKey = PropertiesService.getScriptProperties().getProperty('CLAUDE_API_KEY');
  
  const prompt = `
あなたは経営者の音声メモ分析AIです。
以下の文字起こしテキストから、経営判断に必要な情報を抽出してください。
...（上記プロンプト）

【文字起こしテキスト】
${text}
`;

  const response = UrlFetchApp.fetch('https://api.anthropic.com/v1/messages', {
    method: 'post',
    headers: {
      'Content-Type': 'application/json',
      'x-api-key': apiKey,
      'anthropic-version': '2023-06-01'
    },
    payload: JSON.stringify({
      model: 'claude-sonnet-4-20250514',
      max_tokens: 4096,
      messages: [{
        role: 'user',
        content: prompt
      }]
    })
  });
  
  const result = JSON.parse(response.getContentText());
  return JSON.parse(result.content[0].text);
}
```

**5. テスト（1-2時間）**

a. 実際のPlaud AI文字起こしでテスト
b. 共有フローの確認
c. 権限テスト（社長以外がアクセスできないことを確認）

### アクセス制御

```
Webアプリ URL: https://script.google.com/macros/s/.../exec
  ↓
Google OAuth認証
  ↓
メールアドレスチェック（社長のみ許可）
  ↓
アクセス許可
```

GASコード例:
```javascript
function doGet(e) {
  const userEmail = Session.getActiveUser().getEmail();
  const ownerEmail = PropertiesService.getScriptProperties().getProperty('OWNER_EMAIL');
  
  if (userEmail !== ownerEmail) {
    return HtmlService.createHtmlOutput('<h1>アクセス拒否</h1><p>このアプリは社長専用です。</p>');
  }
  
  return HtmlService.createTemplateFromFile('UI').evaluate();
}
```

## 📌 実装上の注意点

### Claude APIレート制限
- Tier 1: 5 RPM（Requests Per Minute）
- 連続分析時は1リクエスト間隔を15秒以上空ける

### GAS実行時間制限
- 1回の実行: 最大6分
- 長い文字起こし（1時間以上の会議）は分割処理が必要

### データ容量
- Google Sheets: 1ファイル最大200万セル
- 文字起こし1件 = 約500セル想定
- 約4,000件まで保存可能
